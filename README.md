# git_ops-k3s-local

## Purpose
Local / lab GitOps repository for **k3s (Kubernetes)**.

This repository is used for:
- learning Kubernetes
- experimenting with monitoring (Prometheus / Grafana)
- testing GitOps workflows

## What this repository is NOT
- ❌ production repository
- ❌ VM / Docker Compose configuration
- ❌ application source code

## Safety note
This repository is fully isolated from production.
Changes here have **no impact** on Booking or Insurance running on VM.
